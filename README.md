Часы с функцией мониторинга качества воздуха.
Функция измерения CO2, температуры, влажности, давления, есть базовое измерение яркости света.
Часы синхронизируются по NTP протоколу через WI-FI.
Данные датчиков передаются на сервер Blynk.

Подключение:
----------
дисплей 12864 
RS - D8
R/W - D7
E - D6
катод подсветки - D3
----------
датчик mh-z19
RX - D4
TX - D5
----------
датчик BME-280
SDA - D1
SCL - D2
VCC - 3V3
